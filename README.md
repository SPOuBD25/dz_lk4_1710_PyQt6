# Структура кода библиотеки PyQt6 и описание программы

## Архитектура PyQt6

PyQt6 - это набор Python-привязок для фреймворка Qt6. Приложение состоит из двух окон, каждое из которых образовано от базового класса QWidget.

##  Импорты и зависимости

```python
import sys
from PyQt6.QtWidgets import QApplication, QWidget, QLabel
from PyQt6.QtGui import QPixmap, QFont
from PyQt6.QtCore import Qt
```
### Используемые модули:
- **sys** — для управления основным потоком приложения.  
- **QApplication** — основной объект приложения PyQt6.  
- **QWidget** — базовый класс для всех окон.  
- **QLabel** — элемент интерфейса для отображения текста и изображений.  
- **QPixmap** — работа с изображениями.  
- **QFont** — настройка шрифта.  
- **Qt** — доступ к константам для выравнивания и трансформации изображений.

##  Класс `MainWindow` — окно приветствия

Создает простое окно с приветственным сообщением **«Привет, мир!»** и изображением на фоне.

###  Компоненты класса:

- `__init__()` — конструктор, вызывает метод инициализации.  
- `initializeUI()` — задает размеры и параметры окна.  
- `setUpMainWindow()` — добавляет виджеты (текст и изображение).

###  Поведение:
- Размер окна: **630×640 пикселей**  
- Заголовок: **«Приветствие»**  
- Добавляется крупный текст:
  ```python
  hello_label.setText("Привет Мир!")
  hello_label.setFont(QFont("Times New Roman", 60, QFont.Weight.Bold))
  ```
- Устанавливается фоновое изображение `"завтра решим.png"`, масштабируемое под размер окна.
- При отсутствии файла выводится сообщение об ошибке:
  ```
  Файл завтра решим.png не найден
  ```

##  Класс `MainWindow1` — окно резюме пользователя

Создает окно с более сложной структурой — отображением фона, фотографии и текстовых блоков, имитирующих **резюме**.

###  Компоненты класса:

- `__init__()` — конструктор, запускает интерфейс.  
- `initializeUI()` — задает размеры окна и вызывает метод `setUpMainWindow()`.  
- `createImageLabels()` — добавляет фон и фотографию пользователя.  
- `setUpMainWindow()` — создает и размещает все текстовые элементы.

---

###  Работа с изображениями

Метод `createImageLabels()` выполняет двойную загрузку изображений:

1. **Фоновое изображение:**  
   ```python
   bg_image = "грамота.jpg"
   ```
   Масштабируется под размер окна и располагается ниже всех элементов (`label.lower()`).

2. **Изображение пользователя:**  
   ```python
   fg_image = "важный.jpg"
   ```
   Масштабируется до размера 300×200 пикселей и располагается поверх фона.

##  Система позиционирования

Программа использует **абсолютное позиционирование** элементов с помощью метода `move(x, y)`, что обеспечивает точное управление расположением каждого `QLabel`.

Примеры:
```python
hello_label.move(50, 300)
user_label.move(83, 0)
bio_label.move(10, 260)
```

##  Механизм запуска программы

В блоке `if __name__ == '__main__':` создается экземпляр приложения PyQt6:

```python
if __name__ == '__main__':
    app = QApplication(sys.argv)
    window = MainWindow()   # окно приветствия
    window1 = MainWindow1() # окно резюме
    sys.exit(app.exec())
```

- Создаются два окна.  
- `app.exec()` запускает главный **цикл событий**, который обеспечивает работу интерфейса в реальном времени.

##  Принципы работы PyQt6 в этом проекте

- **Дерево виджетов:**  
  Главные окна (`MainWindow`, `MainWindow1`) — корневые элементы.  
  Все `QLabel` — дочерние компоненты, связанные через `self`.

- **Работа с изображениями:**  
  Используется `QPixmap` для загрузки и `QLabel` для отображения, что обеспечивает удобное управление визуальными ресурсами.

- **Событийная модель:**  
  PyQt6 работает на основе цикла событий (`app.exec()`), что делает интерфейс интерактивным и отзывчивым.
